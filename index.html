Powerpoint to Mp4 Converter



The below is an application where the user can upload a powerpoint and get it back with a voice over text.
This comes from the Google API and uses the Speaker Notes to generate the text to speech.
When this button is clicked, it will generate a mp4 file, featuring a video of the powerpoint with a voice over.
Please use sparingly, as it does cost money for me to run.
